What it is:
A Python tool that parses construction site WhatsApp attendance messages and extracts worker names and hours worked per day.

Why it exists:
Built to automate manual data entry at FAS Construction LTD, where site managers/workers send daily attendance on a groupchat via Whatsapp for multiple jobs.

Current version:
V1 handles clean formatted messages. V2 in progress to handle real-world messy WhatsApp formats
